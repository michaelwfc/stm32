# Inteface

## Power

- 12V:  input
- GND: 
- 5V:  output to STM32 chip
  
## PWM

- ENA1: 电机1 enable PWM
- IN1: motor1 PWM
- IN2: motor2 PWM
- ENA2: motor2 enable PWM
