## 电源

- DV 12V 电源线
- DC-DC电压转换模块： 12V转3.3V/5V/12V

## 水泵开关

- 继电器:  
  用继电器控制水泵出水，python程序给树莓派gpio发送高或低电平，控制继电器闭合、开启，间接控制水泵工作
